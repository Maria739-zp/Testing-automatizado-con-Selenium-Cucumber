# Proyecto de testing automatizado para pagina de prueba Saucedemo
# Descripción del proyecto

Este proyecto consiste en la creación del flujo completo de pruebas funcionales para la plataforma de practicas de Saucedemo.El objetivo principal es asegurar la calidad en los flujos criticos de negocio, como la gestion de usuario.

# Herramientas y tecnologías

Lenguaje de Programación: Java

Entorno de Desarrollo Integrado (IDE): IntelliJ IDEA

Herramienta de Automatización Web: Selenium WebDriver.

Framework de Comportamiento (BDD): Cucumber

Gestor de Dependencias y Construcción: Maven

Gestión de pruebas: Excel

Gestion de defectos: Github issues

Navegadores Google Chrome 

Entorno: Windows 11

# Estructura del repositorio 

Plan de pruebas: estrategia, alcance de las pruebas y criterios de aceptación

Casos de prueba: documentación de  escenarios de pruebas positivos y negativos 

Evidencias: Capturas de pantalla de las pruebas ejecutadas

Reporte de bugs:  documentación de los errores encontrados 

# Alcance de las pruebas 

Módulo de autenticación: registro de nuevos usuarios y validación de credenciales
